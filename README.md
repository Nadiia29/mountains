**High Peaks** - це сучасний лендінг, створений для туристичного та спортивного бренду. Він покликаний надихати користувачів на активний відпочинок у горах, пропонуючи зручний інтерфейс для знайомства з турами та бронювання.

Кінцева мета - передати дух пригод через динамічний візуальний досвід та забезпечити простий шлях до взаємодії з брендом (бронювання).

## 🚀 Демо

Живу версію проекту можна переглянути тут: **[High Peaks Live Demo](https://nadiia29.github.io/mountains/)**

## ✨ Ключові особливості

-   **Паралакс-ефекти**: Створюють відчуття глибини та занурення у гірський пейзаж під час скролу.
-   **Адаптивний дизайн**: Коректно відображається на будь-яких пристроях: від великих моніторів до смартфонів.
-   **Модальне вікно бронювання**: Зручна форма для швидкого надсилання заявки без переходу на іншу сторінку.
-   **Бургер-меню**: Інтуїтивно зрозуміла навігація для мобільних версій.
-   **Чистий та сучасний UI**: Мінімалістичний дизайн, що фокусує увагу на природі та активностях.

## 🛠️ Технології

Проект створено з використанням нативних веб-технологій без сторонніх бібліотек або фреймворків:

-   **HTML5**: Семантична розмітка для кращої SEO та доступності.
-   **CSS3**:
    -   Flexbox та Grid для побудови сітки та вирівнювання.
    -   Медіа-запити для адаптивності.
    -   Ключові кадри (keyframes) для анімацій.
-   **JavaScript (Vanilla ES6+)**:
    -   Маніпуляція DOM.
    -   Обробка подій скролу для паралаксу.
    -   Логіка модального вікна та бургер-меню.
